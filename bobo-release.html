Bobo: kiss static site generator

Mon 4 May 19:50:39 BST 2020

Just released first version of bobo, the static site generator with which this website is done.

The name "bobo" is a parody of the famous hugo where "bobo" means "fool" in Spanish.

I was a bit confused by the fact that even the most simple static site generator was at about 500 lines of code. The core of bobo that actually builds the site (bobo-build and bobo-build-page) is ~ 70 lines of code. Then the other two commands, bobo-init and bobo-run are two utilities that initialise the directory and run a busybox httpd server for convenience (but are not necessary at all).

bobo lets you generate a template and then creates pages with the additional capability to edit blog posts (this whole site, including the blog posts, is generated with it).

You can find the source code on my git repos.
